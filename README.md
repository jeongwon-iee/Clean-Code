# ✨ Clean Code ✨
## 애자일 소프트웨어 장인 정신

### 🔖 이 책에서 다루는 내용
- 좋은 코드와 나쁜 코드를 구분하는 방법
- 좋은 코드를 작성하는 방법과 나쁜 코드를 좋은 코드로 바꾸는 방법
- 좋은 이름, 좋은 함수, 좋은 객체, 좋은 클래스를 작성하는 방법
- 가독성이 높아지도록 코드 형식을 맞추는 방법
- 코드 논리를 흩뜨리지 않고서 오류 처리를 완벽하게 구현하는 방법
- 단위 테스트와 테스트 주도 개발ㅇ르 적용하는 방법
